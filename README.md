# To-Do List Web App

A simple To-Do List application built using HTML, CSS, and JavaScript.

# Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Data stored using localStorage

# Tech Stack
- HTML
- CSS
- JavaScript

# How to Run
1. Download the project
2. Open `index.html` in your browser
